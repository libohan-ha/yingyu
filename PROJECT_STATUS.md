## 已知问题 

### 前端
1. 首次加载单词列表页面可能显示空白（需先输入单词）
2. 复习页面中的例句提取可能不够精确
3. 高亮单词释义弹窗位置在某些情况下可能不准确
4. 历史记录为空时未提供足够明显的提示

### 后端
1. 当 DeepSeek API 返回非预期格式时可能导致解析错误
2. 无 MongoDB 连接时的错误处理可以优化
3. DeepSeek API 调用可能较慢，需添加超时处理
4. 生成的文章中可能漏掉部分单词（取决于 AI 响应）

### 集成问题
1. 前后端同时运行需要开两个终端
2. 无实时数据同步机制

## 性能问题
1. 当单词数量较大时，API 调用时间过长
2. 未实现响应缓存机制
3. 初次加载时间可优化
